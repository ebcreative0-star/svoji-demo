## Accomplishments

- Classifier now recognizes checklist_add_multi for comma/list phrasing like "Pridej fotograf, DJ a catering do checklistu" -- all items in one intent
- Classifier now recognizes budget_add_multi for multi-amount messages like "Fotograf 25000, DJ 15000, kvetiny 8000" -- all budget items in one intent
- addChecklistItems: batch DB insert respecting sort_order sequence, mirrors addGuests pattern
- addBudgetItems: batch DB insert using normalizeBudgetCategory, marks source='ai'
- isLikelyNotesPaste: lightweight heuristic (3+ lines or 300+ chars) runs before every request
- parseNotes: dedicated Haiku call with focused prompt to categorize raw text into checklist/budget/guests/unknown
- executeNotesImport: orchestrates checklist_add_multi + budget_add_multi + guest_add_multi then returns plain-text summary
- Chat pipeline STEP 0 intercepts notes pastes before classifyIntent; skips normal pipeline if import succeeds

## Decisions Made

- Notes import executes immediately without confirmation -- no "should I add these?" step per user decision
- Plain-text summary format in summary string (not markdown tables) since MessageBubble uses whitespace-pre-wrap
- parseNotes returns null on any failure -- falls through to normal classifyIntent, no degradation
- STEP 0 detection is purely heuristic-based (length/line count) to avoid extra API cost on every message
