# Phase 11.1 Plan 02: Chat Persistence, Date Headers & Welcome Message Summary

**Chat re-syncs on tab switch via force-dynamic + useEffect, adds Czech date headers between messages, and shows data-state-aware welcome with import nudge for new users**

## Accomplishments

- BUG-08 fixed: chat no longer loses messages on tab switch -- force-dynamic ensures fresh DB fetch, useEffect re-syncs local state
- DateHeader component renders Czech day labels (Dnes, Vcera, or "12. brezna" format) between messages from different calendar days
- WelcomeMessage replaces static empty state: new users see import nudge, returning users with partial data see targeted guidance, returning users with full data see summary + contextual suggestion
- dataState (checklist/budget/guest counts) fetched in parallel with chat messages server-side and passed as prop

## Decisions Made

- `force-dynamic` is the simplest correct fix for BUG-08 -- avoids any client-side polling or Supabase realtime subscription complexity
- `dataState` made optional (`?`) on ChatInterfaceProps so demo mode path (which doesn't pass it) continues to work without change
- All three WelcomeMessage variants use the same suggestion chip pattern (rose pills, onClick fills input) for consistency
