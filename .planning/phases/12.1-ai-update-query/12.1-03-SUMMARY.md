## Decisions Made

- DashboardClientShell wraps DashboardNav + SearchModal + children so the nav can receive onSearchOpen callback from client-managed state -- avoids prop drilling through server layout
- Highlight animation uses Framer Motion backgroundColor keyframe `['#FEF9C3', 'transparent']` over 1.5s; URL param cleaned via `router.replace(pathname, { scroll: false })` after 2s -- clean URL without reload
- Layout fetches 3 data domains in parallel with Promise.all -- minimal latency cost, one await
- globalIndex counter inside render loop handles arrow-key navigation position tracking across domain-grouped sections
